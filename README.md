# sfa
A set of jupyter notebooks for Saint Petersburg University students on Astrometry: Small Field Astrometry course


Astrometric calibration of relatively small fields of view for optical ground-based telescopes equipped with CCD or CMOS cameras. 
The basic coordinate transformation for this type of astrometry is a transformation from RA, DEC to pixel coordinates using tangentioal coordinates. 

RA, DEC - > tangential coordinates transform can be illustrated with this model https://www.geogebra.org/3d/yav7wz6z
